O projeto consiste em um simples sistema para organizar pedaladas em grupo. Quem sabe isso não desperte seu interesse em fazer alguns pedais com seus amigos?!


## Instalação

Primeiramente clone este repositório

```bash
  git clone https://github.com/Casmei/pedaladas-em-grupo.git
  cd pedaladas-em-grupo
```

Em seguida, baixe as dependências do projeto

```bash
  npm install
```

Pronto, para inicar o projeto no ambiente de desenvolvimento, basta escrever o comando a baixo:
```bash
    npm run start:dev
```
    
## Stack utilizada

**Back-end:** Typescript, Nest.js, TypeORM
